# Clon de calculadora iOS creada en react native 

```TypeScript
 git clone https://github.com/astabaresg/CalculadoraIos.git

 cd CalculadoraIos

 yarn install or npm install

# Para ios
cd ios && pod install

react-native run-ios

# Para android
react-native run-android
```
